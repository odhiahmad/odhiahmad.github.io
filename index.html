Featured Project

The product I'm building end-to-end: backend, web, and mobile.

Flagship · 2024 — Present

Loka Kasir

Founder & Full Stack Developer

A complete Point of Sale (POS) platform for small and medium businesses in Indonesia. Loka Kasir covers the full retail workflow: cashier transactions, inventory, cost-of-goods (HPP/BOM) smart pricing, QRIS payments, offline-first web POS, and QR scan-to-order for restaurant tables — all managed from a single admin dashboard.

Backend API Go, Gin, PostgreSQL, Redis — layered architecture with JWT auth, rate limiting, and role-based access
Mobile App (iOS/Android) Flutter with Riverpod state management, offline-capable, responsive mobile & tablet layouts
Admin Dashboard React, TypeScript, Vite, Zustand — full business management: products, outlets, reports, employees
Landing Page Next.js App Router, fully data-driven content
